5 Essential SVG Tags

1. Tag `<svg>`

Use Case

It is the main container for all graphics. It defines the canvas, width, height, and the coordinate system (viewBox) where vectors will be drawn.

Example Code


<svg width="200" height="200" viewBox="0 0 100 100">
    <!-- Graphic content would go here -->
</svg>

More Information

W3C: Element `<svg>`

2. Tag `<circle>`

Use Case

Used to draw simple circles. It is defined by a center point (cx, cy) and a radius (r).

Example Code


<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

More Information

W3C: Element `<circle>`

3. Tag `<rect>`

Use Case

Draws rectangles. The top-left corner position (x, y), width, and height are specified. It also allows for rounded corners.

Example Code


<svg width="150" height="100">
    <rect x="10" y="10" width="100" height="50" fill="blue" />
</svg>

More Information

W3C: Element `<rect>`

4. Tag `<g>`

Use Case

Used to group multiple SVG elements. It is very useful for applying styles (like stroke color) or transformations (rotate, move) to several objects at once.

Example Code


<svg width="200" height="200">
    <g fill="none" stroke="green" stroke-width="5">
        <circle cx="50" cy="50" r="25" />
        <circle cx="100" cy="50" r="25" />
    </g>
</svg>

More Information

W3C: Element `<g>`

5. Tag `<path>`

Use Case

It is the most powerful tool in SVG. It allows drawing complex shapes using a series of drawing commands (Move, Line, Curve, Close) within the `d` attribute.

Example Code


<svg width="100" height="100">
    <!-- Draws a triangle -->
    <path d="M 50 10 L 90 90 L 10 90 Z" fill="orange" />
</svg>
