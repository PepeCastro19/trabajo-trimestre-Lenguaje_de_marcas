5 Essential CSS Properties

Below are five fundamental tools for modern web design, focusing on layout and aesthetics.

1. Property `display: flex`

Use Case

Activates the **Flexbox** model. It is ideal for one-dimensional layouts (a row or a column). It is commonly used to align items within a navigation bar or to center content vertically and horizontally within a container.

Example Code


.container {
    display: flex;             /* Activates Flexbox */
    justify-content: center;   /* Horizontal centering */
    align-items: center;       /* Vertical centering */
    gap: 20px;                 /* Space between items */
}

W3C: Flexible Box Layout Module

2. Property `grid-template-areas`

Use Case

Part of the **CSS Grid Layout** system. It allows defining the structure of a web page by naming specific areas (like 'header', 'main', 'footer') and visually assigning them in a grid. It is perfect for complex two-dimensional layouts.

Example Code


.grid-container {
    display: grid;
    grid-template-areas: 
        "header header"
        "sidebar main"
        "footer footer";
}

.header { grid-area: header; }
.content { grid-area: main; }

More Information

W3C: Grid Template Areas

3. Property `box-shadow`

Use Case

Adds shadow effects around an element's frame. It is used to create depth, highlight cards against the background, or indicate that a button is interactive when hovering over it.

Example Code


.card {
    background-color: white;
    /* offset-x | offset-y | blur-radius | color */
    box-shadow: 0px 4px 8px rgba(0, 0, 0, 0.2);
    padding: 20px;
}

More Information

W3C: Box Shadow

4. Selector `:root` (Custom Properties)

Use Case

Defines global variables (custom properties) that can be reused throughout the document. It is essential for maintaining consistency in colors, fonts, and sizes, and greatly facilitates the implementation of themes (such as dark mode).

Example Code


:root {
    --main-color: #3498db;
    --base-font: 'Arial', sans-serif;
}

h1 {
    color: var(--main-color); /* Uses the variable */
    font-family: var(--base-font);
}

More Information

W3C: CSS Custom Properties

5. Rule `@media`

Use Case

Allows applying different CSS styles depending on device characteristics, mainly screen width (`min-width` or `max-width`). It is the foundation of **Responsive Web Design**.

Example Code


body {
    background-color: white;
    color: black;
}

/* Styles for small screens (mobile) */
@media (max-width: 600px) {
    body {
        background-color: lightgray;
        font-size: 14px;
    }
}
